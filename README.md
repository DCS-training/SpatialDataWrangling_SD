# Edinburgh Spatial Data Wrangling Tutorial 

[cite_start]This repository contains the necessary files to host the **Spatial Data Wrangling in QGIS** tutorial as a live website[cite: 1, 9]. [cite_start]This project provides a step-by-step guide for students to aggregate vector and raster information into uniform hexagonal bins for statistical analysis[cite: 7, 11].

## 📂 Project Structure

* [cite_start]`index.html`: The primary tutorial website file[cite: 9].
* `Spatial_Data_Wrangling_Tutorial.Rmd`: The source R-markdown for the tutorial content.
* [cite_start]`/images`: (Suggested) Directory to store tutorial screenshots for QGIS workflows[cite: 29, 39, 41].
